Returns & Exchanges

We want you to be completely satisfied with your purchase. If you're not happy with your order, our hassle-free returns policy makes it easy to return or exchange items.

Return Window

30-Day Return Policy

You have 30 days from the date of delivery to initiate a return or exchange.

Eligible Items

✓ Eligible for Return

  • • Unworn clothing and accessories
  • • Items in original packaging
  • • Products with all tags attached
  • • Items not listed as final sale

✗ Not Eligible for Return

  • • Final sale items
  • • Personalized or custom items
  • • Items damaged by misuse
  • • Perishable goods
